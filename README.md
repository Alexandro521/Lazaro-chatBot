# Lazaro chat bot para Whatsapp

### como ejecutar lazaro bot


##### instalar dependencias
```javascript
 //npm
npm install
//pnpm
pnpm add
```
##### compilar el proyecto
```js
npm run build
```
##### ejecutar bot
```js
npm run bot
```
#### vincular cuenta de whatsapp
